# RS2V ban list (managed by the server — edits may be overwritten on save)
#
# This file is owned by the security layer (BanManager). It is rewritten whenever
# a ban is added, removed, or an expired ban is cleaned up, so hand edits are
# best made while the server is stopped.
#
# Format (pipe-delimited), one ban per line:
#   SteamID64|P|<unused>|reason             permanent ban
#   SteamID64|T|<expiryUnixSeconds>|reason  temporary ban (UTC seconds since epoch)
#
# Blank lines and lines beginning with '#' are ignored.
#
# Examples (commented out — uncomment and edit to pre-seed bans):
#   76561198000000010|P||Cheating detected via EAC
#   76561198000000011|T|1893456000|Toxic behavior
