## How to Get Listed

Contributors are listed here after their first merged pull request. To contribute:

1. Read the [Contributing Guidelines](CONTRIBUTING.md)
2. Fork the repository and create a feature branch
3. Submit a pull request
4. After your PR is merged, you'll be added to this list

All forms of contribution are valued — code, documentation, bug reports, feature suggestions, testing, and community support.
